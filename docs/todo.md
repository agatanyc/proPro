## Functionality we would like to offer

### Low-level features

* List all foods having at least a specified *protein content*.
* Count the grams of protein in a *serving* of a specified *food*.
* Count the calories in a unit of a specified food.

### Convenience features

* Calculate the percentage of calories from protein in a specified food.
* Support *meals*.
* Find foods whose names match a specified string; e.g., `search("turkey")`
  might return a set including both Turkey breast and Turkey giblets.
